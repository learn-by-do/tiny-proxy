# tiny-proxy

> 一个简单的快速切换 host、url 的 chrome 插件。

点击[这里获取](./tiny-proxy.crx)，或通过源码编译。

```bash
# 开发构建
npm run dev

# 生产构建
npm build
```

Menu ICONs from [font awesome](https://fontawesome.com/)。
